Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist skeeks/cms-theme-unify-v2 "^0.0.1"
```

or add

```
"skeeks/cms-theme-unify-v2": "^0.0.1"
```
